기숙사별 총정리

세종1관-장점

- 편의점, 세탁소, 카페 있음
- 방이 기숙사 3개중에서 젤 넓음
- 방음이 잘됨
- 냉장고 있음
- 층마다 분리수거함&정수기 있음
- 최근에 방 리모델링해서 깨끗한 편
- 기숙사비 제일 쌈

세종1관-단점

- 외국인이 너무 많음(안내방송 3개국어)
- 화장실이 3개 기숙사 중에서 젤 좁음
- 엘베 자주 고장남
- 편의점이 있지만 굉장히 좁음
- 공용 공유기가 없음
- 밥 신청이 필수인데 맛이 없음

세종2관-장점

- 시설이 깨끗함
- 층마다 분리수거함 있음
- 공유기 있음
- 2층에 휴게실 있고 티비있음
- 냉장고 있음
- 빨래 건조대 있음
- 외국인 없음
- 전자레인지 있음

세종2관-단점

- 방음 안좋음
- 편의시설 없음
- 짝수층에만 정수기 있음
- 세종1관 보다 기숙사비 조금 더 비쌈

세종 2관은 두드러지는 장점도 단점도 없이 무난
밥 맛없음

행복기숙사-장점

- 외국인 없음
- 젤 최신 기숙사 (2017년에 완성)
- 밥이 3개 긱사 중에 젤 맛있음
- 밥 신청한거 만큼 못먹으면 간식같은걸로 바꿔주는 제도 있음
- 프린트 가능한 시설 있음
- 편의시설 좋음(편의점 넓고,카페)
- 세종은 세탁기 1300원인데 여긴 1000원
- 전자레인지 있음
- 화장실 젤 넓음
- 룸메 신청 가능
- 스터디룸 짝수층마다 있음
- 와이파이 있음
- 전신거울 있음
- 한번 들어가면 연장제도 있어서 계속 살 수 있음
- 헬스장 무료이용
- 복도에 휴게실 있음

행복기숙사-단점

- 방음이 케바케로 안좋음
- 냉장고가 없음
- 주말엔 세종이랑 밥 비슷
- 방이 가장 좁음
- 건조대 없음
- 기숙사비 젤 비쌈
- 우리학교 학생이 많지만 타대학 학생이 룸메로 걸릴수도 있음
- 층마다 분리수거함 없어서 1층 내려가야함

광개토관-용당캠퍼스_광개토관-A동 : 지어진지 꽤 됨

- 호실내에 화장실이 없음
- 2인실의 호실 4개가 하나의 큰 거실에 들어있는 이중실내구조이고
호실 4개 공용으로 냉장고가 거실에 하나 있음
- 화장실은 옆 거실 사이에 있어서 16명이 공용으로 사용
- A동은 엘리베이터가 없어서 5층까지 계단 사용

광개토관-용당캠퍼스_광개토관-B동 : 지어진지 꽤 됨

- 호실 내에 화장실이 없음
- B동의 경우엘리베이터가 있음
- 2인 1실 방 세 개가 한 호실이며, 두 호실이 한 화장실을 사용함
- 화장실이 큼
- 기숙사 식당과 연결된 통로가 있어 비 안맞고 이동가능
